💻 第二步：连接你的 VPS 服务器

购买成功后，你会收到 IP、用户名（通常是 root）和密码。

  1. 下载 SSH 工具：推荐使用 Tabby 或 Termius。
  2. 连接命令：
    在终端输入 ssh root@你的服务器IP。
  3. 首次登录：如果是第一次连接，输入 yes 并回车，然后输入密码（注意：密码输入时不会显示任何字符）。
